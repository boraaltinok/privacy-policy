## Getting Started

Translate Picture is an AI-powered mobile application that provides instant translation of text from images. Users can capture or upload images containing foreign text and receive immediate, accurate translations in their preferred language. The app is designed for travelers, expatriates, language learners, and professionals working in multilingual environments who need quick understanding of written content in images. Using advanced OpenAI Vision API technology, the app performs both text extraction and translation in a single operation, eliminating complex setup and delivering results in seconds. The app is available for iOS devices.

### Key Features
- **Image Capture And Upload For Translation**: Easily capture or upload images for translation.
- **Target Language Selection With Auto-detection**: Automatically detect the target language or choose it manually.
- **Optional Source Language Specification**: Specify the source language if needed.
- **AI-powered Text Extraction And Translation From Images**: Utilize AI technology for accurate text extraction and translation.
- **Copy And Share Translated Text Functionality**: Copy and share translated text seamlessly.
